AI -> 啊
O ->哦
E -> 诶
U -> 油
TH -> thank you的开头 th
etc -> 代替C D G K N R S T H Y Z
WQ -> will的开头 wi 或者 我
MBP -> 就是mbp的口型
FV -> f和v的口型

一共9个口型，
一个单词的一个字符对应一个口型。
例子：  l   o   v   e
           |    |    |    |
         L  AI  AI  FV
如果这个字符没有口型就用上一个字符的口型，
如果有的是一串字符对应一个口型同理。
例子：t h o u g h t
          |  |  |  |  |  |  |
      TH {      O       }
你也可以对照Map里的口型来编写，
然后再PhrasesMap里按照；
例子：
  "短语" {
        "0":"口型"
        "1":"口型"
        "2":"口型"
        ....
        },

这样的格式填写下去，
并上传到"enqi口型对照文件夹里"。
